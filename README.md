# SHINE ON JEWELRY — Free Website

This is a free, mobile-friendly jewelry store starter.

## Included
- Luxury homepage
- Product search and category filter
- Shopping cart
- Demo checkout
- Simple admin dashboard
- Browser-based product and order storage
- Responsive mobile design

## Open it
1. Unzip the folder.
2. Open `index.html` in Chrome, Safari, or Edge.
3. Open `admin.html` to add products and view demo orders.

## Publish for free with GitHub Pages
1. Create a new GitHub repository.
2. Upload all files from this folder.
3. Open repository **Settings → Pages**.
4. Under **Build and deployment**, select **Deploy from a branch**.
5. Choose `main` and `/root`, then save.
6. GitHub will give you a public website address.

## Important
This free version uses browser `localStorage`.
- Products and orders are visible only in the same browser/device.
- It does not process real payments.
- For a real online store later, connect Supabase/Firebase and a payment provider.

## Customize
- Change product examples in `data.js`.
- Change shop text and contact links in `index.html`.
- Change colors in `styles.css`.
